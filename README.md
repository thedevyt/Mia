# MIA – The AI Adapter for Linux Mint

Speak naturally. Your system understands and executes your commands.

### Features
- Natural language → real actions (files, system, processes, packages, media, network, Git, Docker)
- Dual LLM engine: local **Ollama** or **OpenAI** (auto‑select)
- YAML intent mapping with Jinja variables
- Modular plugin system
- CLI (Typer), GUI (Tkinter), Voice (Vosk / Whisper)
- Command preview + confirmation
- Session memory and colorful logs (Rich)
